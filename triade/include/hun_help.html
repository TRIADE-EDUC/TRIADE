Segítség

Összes lehetõség a file manager-ben.

Mindíg az aktiális mappádat láthatod. A következõ példa szerint: toto/aaaa/dfg
Az információból láthatod, hogy te most a "dfg" mappába dolgozol. To go back to a parent directory, click on its name.

You can select the language after clicking on this picture.

Ha ráklikkelsz a frissítés ikonra, az oldaladba újra be lesznek olvasva az információk

Ez az ikon a program dokumentációjára fog elvinni

Ha erre az ikonra kattintasz, kilépsz és bejön az alpértelmezett oldalad.

A táblázatban megtalálod a "File neve, mérete..." információkat
Ha ráklikkelsz valamely meghatározásra, akkor azok szerint fogja rendezni a file-jaidat

Ennél az ikonnál a partner mappába fogsz átjutni .
Ezek az ikonok azt jelentik, hogy mappák és ezekbe file-ok vannak, vagy lehet tárolni,
Ha egy mappára ráklikkelsz, anna a tartalmába fogsz jutni, ezeket a file-okat le is töltheted, ha vissza kívánsz jutni az elõzõ mappába, klikkel a "Vissza" nyomógombra.

Ennek az ikonnak a segítségével másolhatsz. Válaszd ki azt a mappát ahova másolni akarod, majd nyomj az OK nyomógombram ekkor megtörténik a másolás

Ezzel az ikonnal file-okat mozgathatsz át. Ez a mûvelet nem egyezik meg a másolással, mivel itt az eredeti file törlésre kerül.

Ezzel az ikonnal átnevezhetsz file-okat. Be kell írnod a file teljes nevet majd azt, amire át kívánod nevezni és ráklikkelni az átnevezés nyomógombra, majd használd a "vissza" gombot.

Ha erre az ikonra klikkelsz, akkor a melette lévõ file-t törölni fogod. Ha sikerült törölni, a következõ üzenet jön vissza: " File törölve".

Itt editálhatod a text-ben írt file-okat. Kilikkelj rá, majd a file be fog kerülni egy text box-ba, itt végrehajthatod az editálást, majd nyomj a mentés gombra, ha nem akarod a módosítást menteni, nyomj a "mégsem" gombra.

File feltöltése
File feltölétése a saját gépedrõl. Válaszd ki azt a mappát ahová fel akarsz tölteni, majd nyomj a "tallózás" gombra, ekkor kiválaszthatod az a file-t, amit fel akarsz tölteni. Nyisd meg, majd amikor a file elérési útvonala bekerült a feltöltõ mezzõbe, nyomj a "feltöltés" gombra, ha túl nagy file-t töltesz fel, nem fog megjelenni az nyugtázó oldal

Új mappa létrehozása
Válaszd ki azt a mappát, ahová létre kívánod hozni az új mappát, majd írd be az "új mappa létrehozása" box-ba a létrehozni kívánt mappa nevét, ha ez megvan, nyomj a "mappa létrehozása" gombra.

Új file létrehozása
Válaszd ki, hiogy hova kívánod létrehozni a file-t, a következõ kiterjesztéseket hozhatod létre: CGI, TXT, HTM, HTML... ( Hozzáadhatsz további engedélyeket is).
Írd be a szövegmezõbe a létrehozni kívánt file nevét, majd nyomj a "Létrehoz" gombra
