AIDE

Le chemin de répertoires affiché en haut à gauche, par exemple : toto/aaaa/dfg
vous indique dans quel dossier vous êtes actuellement. Dans cet exemple, les fichiers du répertoire "dfg" sont listés. Pour revenir dans un répertoire parent, cliquez sur son nom.

Vous pouvez sélectionner la langue en cliquant sur cette image.

Pour actualiser la page courante, cliquez sur cette icône et la page sera rafraichie.

Cette icône permet d'afficher l'aide.

Pour vous déconnecter, cliquez sur cette icône. Votre session sera achevée et vous serez redirigé vers la page d'entrée.

Dans la première ligne du tableau principal, vous trouvez : "Nom du fichier - Taille - Type..."
Cliquez sur ces descriptions pour classer les fichiers dans l'ordre correspondant. Recliquez et vous inverserez l'ordre.

Dans la colonne "Nom du fichier", les fichiers et dossiers du répertoire courant sont listés. Pour remonter dans le dossier parent, cliquez sur l'icône .
En cliquant sur un dossier, vous listerez son contenu.
En cliquant sur un fichier, celui-ci s'affichera ou vous sera proposé en téléchargement s'il ne s'agit pas d'un fichier texte ou exécutable par le serveur. Aprés l'affichage, pour revenir à la liste des fichiers, cliquez sur le bouton "Précédent" de votre navigateur.

Cette icône vous permet de copier un fichier dans un autre répertoire. Pour le copier, vous devrez sélectionner le répertoire de destination et cliquer sur "OK". Vous pourrez annuler cette action à tout moment en cliquant sur "Annuler".

Cette icône vous permet de déplacer un fichier dans un autre dossier. La méthode est identique à la fonction "copier" mais le fichier source sera supprimé aprés la copie.

Cette icône vous permet de renommer un fichier ou un dossier. Il suffit de saisir le nom complet dans la zone de texte et de cliquer sur "Renommer". Vous pouvez revenir à la liste en cliquant sur "Retour" si vous ne voulez plus renommer le fichier ou le dossier.

Pour supprimer un fichier, cliquez sur cette icône. Une confirmation vous sera demandée.

Pour modifier un fichier texte, une page HTML ou autre, cliquez sur cette icône. Le contenu du fichier sera éditable dans une zone de texte. Pour valider les modifications, cliquez sur "Enregistrer" ou "Annuler" pour revenir à la page précédente.

Transférer un fichier
Pour transférer un fichier de votre disque dur vers le site, cliquez sur "Parcourir" et sélectionnez un fichier. Cliquez ensuite sur "Transférer"... et patientez le temps du transfert. Les fichiers lourds ne sont pas conseillés. Les caractères de fin de ligne pour les fichiers ASCII sont remplacés par des \n.

Créer un nouveau répertoire
Dans la zone de texte prévue à cet effet, saisissez un nom de répertoire. Celui-ci ne doit pas contenir de caractères spéciaux comme les espaces, accents ou autre. Ces caractères seront supprimés automatiquement si vous les saisissez. Cliquez sur "Créer", le dossier sera créé dans le répertoire courant.

Créer un nouveau fichier
Vous avez la possibilité de créer des fichiers vierges portant les extensions : CGI, TXT, HTM, HTML, etc... (vous pourrez ensuite les modifier en les éditant). Saisissez un nom de fichier valide, cliquez sur "Créer" et celui-ci sera créé automatiquement dans le répertoire courant.
S'il s'agit d'un fichier HTM ou HTML, un squelette HTML sera généré.
