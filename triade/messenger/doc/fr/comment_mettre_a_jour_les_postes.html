Instructions

  • Pour mettre à jour les postes clients IntraMessenger

Description

Le fichier UDS-Update.exe est indispensable dans le répertoire de l'exécutable (IntraMessenger.exe).
Utilisateurs de Windows Vista : le fichier UDS-Update.exe.manifest est indispensable dans le répertoire de l'exécutable (la désactivation de l'UAC peut être nécessaire).
La mise à jour sera vérifiée 30 minutes après le démarrage.

Deux méthodes possibles :

  • en automatique via internet (depuis le site officiel) [1]
  • via le serveur [2]

[1] Mises à jour via internet (depuis le site officiel)

Sur chaque poste client, dans les options, il suffit de cocher "Depuis internet", et les mises à jour s'effectueront automatiquement.

L'option serveur _FORCE_UPDATE_BY_INTERNET permet de forcer les postes clients.
Les postes client ne pourront pas désactiver la mise à jour.

[2] Mises à jour via le serveur (local/intranet ou internet)

  • Placer dans le répertoire /distant/update/ du serveur la dernière version du client IntraMessenger.exe.
  • Dans le fichier /distant/update/version.ini à la ligne :
    version=
    y indiquer la version du client.

    Exemple :
    version=1.21A
L'option serveur FORCE_UPDATE_BY_SERVER permet de forcer (bloquer) la mise à jour.
Les postes client ne pourront pas désactiver la mise à jour, ni choisir la mise à jour via internet (site officiel).

Si de nouvelles traductions sont disponibles (fichiers XX.ini), les placer dans le répertoire /distant/update/ du serveur.

Connaitre le numéro de version de l'exécutable client
