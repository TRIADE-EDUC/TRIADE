Septembre 2013 : IntraMessenger serveur 2.0.5

  • Accès ACP : gestion multi-administrateurs : chacun d'eux pouvant avoir des droits d'accès différents.
    Pour ceux qui ne souhaitent pas activer cette fonctionnalité, voir l'option serveur : _ACP_PROTECT_BY_HTACCESS.
    Option serveur : _ACP_ALLOW_MEMORY_AUTH : autoriser (ou non) la mémorisation des identifiants (dans les cookies).
  • Sauvegarde des fichiers utilisateurs :
    - Option serveur : _BACKUP_FILES : autoriser (ou non) la sauvegarde des fichiers.
    - Option serveur : _BACKUP_FILES_THIS_LOCAL_FOLDER_ONLY : sauvegarder uniquement ce répertoire local (vide : choix par l'utilisateur).
    - Option serveur : _BACKUP_FILES_ALLOW_MULTI_FOLDERS : autoriser la sauvegarde de plusieurs répertoires.
    - Option serveur : _BACKUP_FILES_ALLOW_SUB_FOLDERS : autoriser la sauvegarde (récursive) des sous-dossiers.
    - Options serveur :
      _BACKUP_FILES_MAX_NB_ARCHIVES_USER - _BACKUP_FILES_MAX_ARCHIVE_SIZE - _BACKUP_FILES_MAX_SPACE_SIZE_USER - _BACKUP_FILES_MAX_SPACE_SIZE_TOTAL -
      _BACKUP_FILES_FTP_ADDRESS - _BACKUP_FILES_FTP_LOGIN - _BACKUP_FILES_FTP_PASSWORD_CRYPT - _BACKUP_FILES_FTP_PORT_NUMBER - _BACKUP_FILES_FORCE_EVERY_DAY_AT -
  • Option serveur : _ALLOW_HIDDEN_STATUS : autoriser (ou non) l'usage de l'état caché (indiqué "non connecté" pour tous).
  • Option serveur : _ALLOW_HISTORY_MESSAGES_EXPORT : autoriser (ou non) l'utilisateur à exporter (au format Excel) l'historique des messages archivés.
  • Option serveur : _FORCE_OPTION_FILE_FROM_SERVER pour activer l'envoi du fichier de config par défaut (options.ini) aux postes clients (client 1.31D mini).
  • Option serveur : _SHARE_FILES_ALLOW_ACCENT : autoriser (ou non) l'utilisation d'accents dans les fichiers
    (renomme automatiquement les fichiers qui sont envoyés sans accents).
    Désactivée uniquement si serveur FTP sous Windows (exemple : FileZilla Server) !
  • Rôle utilisateurs : ROLE_OFFLINE_MODE : en mode entreprise, permet d'avoir IntraMessenger installé (donc fonction inventaire du parc informatique)
    sans que l'utilisateur n'ai à l'utiliser (ex: parce qu'il ne le souhaite pas).
    Permet sinon de bannir temporairement des utilisateurs...
  • Rôle utilisateurs : ROLE_CHANGE_SERVER_STATUS : permet de changer directement depuis le client l'état des serveurs/services.
  • Rôle utilisateurs : ALLOW_HIDDEN_STATUS.
  • ACP : statistiques :
    - Nouveaux graphiques : messages ShoutBox - Fichiers publiés - Fichiers échangés.
    - Corrections.
  • ACP : tableau de bord : affichage de l'espace de stockage utilisé (avec jauge horizontale) pour le partage et la sauvegarde de fichiers.
  • ACP : informations utilisateur : affichage de l'espace de stockage utilisé (avec jauge horizontale) pour le partage et la sauvegarde de fichiers.
  • ACP : informations utilisateur : liste des groupes auxquels appartient un utilisateur.
  • ACP : informations utilisateur : possibilité de modifier le genre (si profil verrouillé).
  • ACP : Liste des comptes périmés.
  • ACP : Messagerie administrateur (informations vers client) : texte plus long, sur plusieurs lignes.
  • ACP : menus réorganisés.
  • ACP : options réorganisés.
    - Pour chaque options majeure : si non activée, les options dépendantes sont grisées (mise à jour de /common/styles/*.css).
    - Onglet partage de fichiers : ne pas afficher toutes les options d'un coup (pour pouvoir en afficher plus).
    - Possibilité de mettre des apostrophes dans certaines options (_SITE_TITLE - _SITE_TITLE_TO_SHOW - _SCROLL_TEXT - _AWAY_REASONS_LIST - _BUSY_REASONS_LIST...).
  • ACP : affichage de l'heure (fuseau horaire) de PHP et de MySQL (indiquer si paramètrage différent).
  • Nouvelles possibilités d'authentification externe : Fire Soft Board - ProjeQtOr
  • Option serveur : _GROUP_ID_DEFAULT_FOR_NEW_USER.
  • Option serveur : _STOP_USE_THIS_SERVER_ADDRESS_NOW_USE_THIS_URL_AFTER_LOGIN.
  • Renommage de certaines options serveur (simplement enregistrer les options pour appliquer) :
    • _ALLOW_INVISIBLE -> _ALLOW_HIDDEN_TO_CONTACTS
    • _ALLOW_USER_TO_HISTORY_MESSAGES -> _ALLOW_HISTORY_MESSAGES
    • _ALLOW_USER_RATING -> _ALLOW_CONTACT_RATING
    • _DISPLAY_USER_FLAG_COUNTRY -> _FLAG_COUNTRY_FROM_IP
    • _LOG_MESSAGES -> _HISTORY_MESSAGES_ON_ACP
    • _OUTOFDATE_AFTER_X_DAYS_NOT_USE -> _OUTOFDATE_AFTER_NOT_USE_DURATION
    • _FULL_CHECK -> _SLOW_NOTIFY
    • _LOCK_USER_CONTACT_LIST -> _ALLOW_MANAGE_CONTACT_LIST (valeur inversée !)
    • _LOCK_USER_OPTIONS -> _ALLOW_MANAGE_OPTIONS (valeur inversée !)
    • _LOCK_USER_PROFILE -> _ALLOW_MANAGE_PROFILE (valeur inversée !)
    • _HIDE_COL_FUNCTION_NAME -> _ALLOW_COL_FUNCTION_NAME (valeur inversée !)
