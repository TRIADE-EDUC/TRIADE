How to

  • Update IntraMessenger clients

Description

The file UDS-Update.exe must be in exe directory.
Windows Vista users : the file UDS-Update.exe.manifest must be in exe directory (and maybe disable UAC).
Update will be checked 30 minutes after startup.

Two ways:

  • automatic by internet (from official website) [1]
  • by the server [2]

[1] Internet updates (by official website)

On clients, in options screen, click "From internet", and updates will be automatics.

Server option _FORCE_UPDATE_BY_INTERNET allow to force client updates. Clients cannot disable updates.

[2] Server updates (local/intranet or internet server)

  • In /distant/update/ server directory, put last client version IntraMessenger.exe.
  • In file /distant/update/version.ini at row :
    version=
    write the client version.

    Example:
    version=1.21A
Server option FORCE_UPDATE_BY_SERVER allow to force client updates.
Clients cannot disable updates or choise update from internet (site officiel).

If new translations are available (files XX.ini), upload them on server folder /distant/update/.
