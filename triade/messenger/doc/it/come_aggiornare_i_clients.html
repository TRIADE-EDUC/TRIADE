Come aggiornare i 'clients' di IntraMessenger

Descrizione

Dalla versione del 'client' 1.0.4.13 (1.04N) (e server 1.3.2.108), l'aggiornamento dei 'clients' è possibile.
Il file (eseguibile) UDS-Update.exe è indispensabile che sia presente nella cartella dell'eseguibile. (/programmi/IntraMessenger/)
Per gli Utenti di Windows Vista : il file UDS-Update.exe.manifest è indispensabile che sia presente nella cartella dell'eseguibile.
L'aggiornamento sarà verificato dopo 30 minuti l'inizio dell'esecuzione.

Due metodi possibili : in automatico via internet (tramite il sito ufficiale), o via il server.

Aggionamento via internet (tramite i site ufficiale)

In ogni 'client', nelle opzioni, è sufficiente premere il pulsante radio "Da internet",
e gli aggiornamente saranno effettuati automaticamente.

Aggionamento via server (locale/intranet o internet)

- Mettere nella cartella /distant/update/ del server l'ultima versione del 'client' client IntraMessenger.exe.
- Indicare nel file /distant/update/version.ini alla riga :
version=
indicare la versione del 'client' copiato (nella cartella indicata quì sopra).
Es : version=1.06H

Esiste la possibilità sul server di forzare (bloccare) l'aggiornamento grazie all'opzione : _FORCE_UPDATE_BY_SERVER.
I 'clients' non potranno impedire l'aggiornamento, nè scegliere l'aggiornamento via internet (tramite il sito ufficiale).
