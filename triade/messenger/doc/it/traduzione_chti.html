Guida di installazione per IntraMessenger (server)

Traduzioni chtimi (ch'ti) in diretta :-

Eì possibile attivare la traduzione (di alcune parole) in Chti, per far ciò :
- aprire il file /distant/msg_send.php
- togliere il carattere di commento (#) all'inizio delle righe (160 e 161 circa) :
    #include ("../common/words_chti.inc.php");
    #$msg = textChti($msg);
- registrare le modifiche (e transferirle sul server ove necessario) .

Per modificare il dizionario chti, modificare il file /common/words_chti.inc.php

Funziona solo se il cryptaggio dei files non è attivo (c_option_crypt_msg).

Da notare : per le versioni scaricate prima del 27/04/2008, la traduzione ha luogo sul server immediatamente prima dell'invio verso il 'client' destinatario, pertanto l'autore vede il suo testo nella versione originale (non modifificato).
