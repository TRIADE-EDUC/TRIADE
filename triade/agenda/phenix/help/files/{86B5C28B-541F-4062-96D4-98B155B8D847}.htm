Dénomination et type de fichiers

Les Mods doivent être nommés comme ceci:

  • commencer par «Mod_Phenix_VX_», X représentant le numéro de version de Phenix
  • se terminer par le nom du Mod sans caractères spéciaux (espaces, accents, guillemets, ...)
  • suivi de l'extension ".txt"

Exemple :


Mod_Phenix_V5_Scission_de_note_recurente.txt


Les Mods peuvent être fournis aux utilisateurs sous deux formes différentes :

  • directement  au format texte
  • au format zip avec compression normale si le Mod inclus des fichiers annexes
